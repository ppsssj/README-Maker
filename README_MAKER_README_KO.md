# Readme Maker

README를 WYSIWYG 방식으로 더 쉽고 구조적으로 작성할 수 있도록 지원하는 VS Code 확장 프로그램입니다.

## 개요

**Readme Maker**는 개발자가 Markdown 문법을 한 줄씩 직접 작성하지 않아도 README를 보다 쉽게 만들 수 있도록 설계된 VS Code 확장 프로그램입니다.

기존의 raw Markdown 편집 방식에 의존하는 대신, 보다 시각적이고 구조적인 편집 흐름을 제공하는 것을 목표로 합니다. 사용자는 제목, 본문, 리스트, 이미지, 코드 영역 등의 블록을 조합해 README를 구성할 수 있고, 확장 프로그램은 이를 바탕으로 정리된 Markdown 결과물을 생성합니다.

목표는 명확합니다. Markdown 작성의 진입 장벽을 낮추고, 더 읽기 좋고 완성도 있는 프로젝트 문서를 더 빠르게 만들 수 있도록 돕는 것입니다.

## 왜 Readme Maker인가

좋은 README는 프로젝트 완성도를 크게 좌우하지만, 실제 작성 경험은 비효율적인 경우가 많습니다.

대표적으로 다음과 같은 문제가 있습니다.

- 원본 편집 화면과 미리보기 화면을 반복적으로 오가야 함
- Markdown 문법을 직접 기억하고 입력해야 함
- 섹션 구조를 정리하는 과정이 번거로움
- 프로젝트 소개, 기능 설명, 사용법, 스크린샷 정리 과정에서 반복 작업이 많음

Readme Maker는 이러한 문제를 해결하기 위해 보다 직관적이고 구조적인 작성 흐름에 집중합니다.

## 핵심 아이디어

이 확장 프로그램의 핵심은 **블록 기반 README 작성 워크플로우**입니다.

사용자는 다음과 같은 블록을 조합해 문서를 작성할 수 있습니다.

- 제목
- 본문
- 리스트
- 체크리스트
- 코드 블록
- 이미지
- 구분선

각 블록은 시각적으로 배치되고 관리되며, 최종적으로는 GitHub와 호환되는 일반 `.md` 파일로 변환됩니다.

즉, 작성 경험은 더 직관적으로 만들고, 결과물은 기존 Markdown 생태계와 호환되도록 유지하는 방식입니다.

## 주요 기능 계획

- 블록 기반 README 편집
- WYSIWYG 방식의 작성 흐름
- 자동 Markdown 생성
- 미리보기 친화적인 편집 경험
- 섹션 순서 재정렬
- 이미지, 코드 블록, 체크리스트 지원
- GitHub 호환 Markdown 출력

## 활용 대상

- 프로젝트 README를 자주 작성하는 개발자
- 포트폴리오 저장소 문서를 정리하려는 사용자
- 해커톤, 캡스톤, 사이드 프로젝트 문서를 빠르게 정리하려는 경우
- VS Code 안에서 바로 문서를 작성하고 싶은 사용자
- Markdown 문법보다 시각적인 편집 흐름을 선호하는 사용자

## 비전

Readme Maker는 단순한 Markdown 미리보기 도구가 아닙니다.

Markdown 호환성은 유지하면서도, 보다 직관적인 작성 경험을 제공하는 실용적인 README 작성 도구를 목표로 합니다.
